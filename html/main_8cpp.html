◆ main()

int main ( )

Processes the random requests from various servers (the loadbalancer)

Takes in user input for the number of servers and the amount time to run the loadbalancer. Creates a full queue of requests based on the number of servers * 2. Then creates the webservers and adds requests to each server. We then have a for loop that iterates until the current time is greater than or equal to the clock time. Within the loop a random server is selected and if the current time is greater than the start time + processing time then the request will be processed and displayed.
